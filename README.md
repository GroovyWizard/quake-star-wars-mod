# quake-star-wars-mod
Simple Star Wars mod for QUAKE 1.
In `source` folder you will find all the source code, textures and models, use as you wish.

<p> <strong> v0.1.0  </strong> </p>

Features:
- Lightsaber

Planned Features:
- Enemy Reskin (Troopers)
- Enemy Reskin (Droids)
- Change Lightsaber Colors
- Change Weapons
